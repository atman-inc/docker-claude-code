# Docker Claude Code

Node.js v22、SSH、Git をサポートした Claude Code 用の Docker イメージです。

## 機能

- Node.js v22 ランタイム環境
- SSH クライアント・サーバーサポート
- Git バージョン管理
- Claude Code AI アシスタント統合

## イメージのビルド

```bash
docker build -t claude-code .
```

## コンテナの実行

```bash
docker run -it --rm claude-code
```

## 要件

- Docker
